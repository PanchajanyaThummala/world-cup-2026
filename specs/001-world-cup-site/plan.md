## Summary

Build a single-page, statically hosted React + Vite + TypeScript fan website for the FIFA
World Cup 2026. The site delivers five major sections — cinematic hero, groups/venues, nostalgia
timeline, cultural impact, deep facts, and a bracket/standings tracker — all powered by local
TypeScript data files (no backend). Deployed to Vercel. Animations via Framer Motion; styling
via Tailwind CSS v4 with custom gold/dark design tokens.

## Animation Architecture

All animations are centralized in `src/lib/motion.ts` to ensure consistency:

```
fadeInUp:    y: 20→0, opacity: 0→1, duration: 0.5s, ease: easeOut
fadeIn:      opacity: 0→1, duration: 0.4s
staggerContainer: staggerChildren: 0.08s
scaleIn:     scale: 0.95→1, opacity: 0→1, duration: 0.4s
slideInLeft: x: -30→0, opacity: 0→1, duration: 0.5s
slideInRight: x: 30→0, opacity: 0→1, duration: 0.5s
```

All scroll-triggered animations use Framer Motion's `whileInView` with `viewport: { once: true }`.
Reduced motion: all variants collapse to `opacity` only when `prefers-reduced-motion: reduce`.

## Data Architecture

All content lives in `src/data/` as typed TypeScript constant arrays. No fetching, no async.

| File | Records | Notes |
|------|---------|-------|
| `teams.ts` | 48 teams | name, code (e.g. "BRA"), flag emoji, group, confederation |
| `groups.ts` | 12 groups | A–L, 4 teams each, standings zeroed for pre-tournament |
| `venues.ts` | 16 venues | 11 USA + 2 Canada + 3 Mexico |
| `schedule.ts` | 104 matches | group stage 72 + knockout 32 |
| `moments.ts` | ≥15 entries | year, host, title, narrative, era |
| `impact-stories.ts` | ≥6 entries | year, title, narrative, era badge, category |
| `facts.ts` | ≥10 entries | hook, supporting text, category |
| `bracket.ts` | 63 slots | Round of 32 → Final, team names or TBD |
